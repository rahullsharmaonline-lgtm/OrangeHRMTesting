# OrangeHRM Testing Project

## 📌 Project Overview

This project demonstrates manual testing of the OrangeHRM web application.
The testing includes functional, validation, and boundary testing with proper documentation.

## 🔍 Modules Tested

* Login Module
* Dashboard
* Employee Management (PIM)
* Admin Module

## 🧪 Testing Types Performed

* Functional Testing
* Validation Testing
* Boundary Testing
* UI Testing

## 🐞 Bugs Identified

* Missing input validation for long username
* Loading delay issue in Add Employee module
